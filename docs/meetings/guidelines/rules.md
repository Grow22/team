# 📋 7팀 회의 룰

---

## 기본 사항

- 회의는 **매주 화요일** 대면으로 진행한다.
- 서기는 회의 직전 **사다리 타기**로 결정한다.
- 서기는 회의록 템플릿을 복사하여 해당 주차 회의 내용을 기록한다.
- 작성된 회의록은 `docs/meeting/` 폴더에 `YYYY-MM-DD-meeting.md` 형식으로 저장한다.

---

## 회의 진행 순서

1. 서기 선정 (사다리 타기)
2. 지난 주 회고 (좋았던 점 / 어려웠던 점)
3. 지난주 할 일 체크
4. 이번 주 논의 안건
5. 다음 주 할 일 분배
6. 기타 메모 및 마무리
